# 华夏百味图 | 八大菜系数据可视化

## 📖 项目简介

这是一个整合了8个独立可视化页面的全屏滚动单页应用，展示中国八大菜系的多维度数据分析。

## 🎯 功能特点

- ✨ **全屏滚动体验**：使用fullPage.js实现流畅的页面切换
- 🖱️ **多种交互方式**：支持鼠标滚轮、键盘方向键、触摸滑动
- 🎨 **统一视觉风格**：中国传统宣纸风格，朱砂红配色
- 📊 **8个可视化页面**：涵盖历史、地域、口味、菜品等多个维度

## 🚀 使用方法

### 方法一：本地预览

1. 直接双击打开 `index.html` 文件
2. 使用现代浏览器（推荐Chrome/Edge/Firefox最新版）

### 方法二：本地服务器（推荐）

使用Python快速启动HTTP服务器：

```bash
# 进入项目目录
cd 八大菜系可视化-部署版

# Python 3
python -m http.server 8000

# 然后在浏览器访问：http://localhost:8000
```

或使用Node.js的http-server：

```bash
npx http-server -p 8000
```

### 方法三：部署到服务器

将整个文件夹上传到Web服务器（Nginx/Apache）的根目录即可访问。

## 🛠️ 技术栈

- **fullPage.js 4.0.20** - 全屏滚动框架
- **ECharts 5.4.3** - 数据可视化图表库
- **GSAP 3.12.2** - 动画库
- **Tailwind CSS** - 原子化CSS框架
- **React 18** - 部分页面使用React组件
- **原生Canvas** - 自定义动画效果

## 📌 注意事项

1. **网络依赖**：所有外部库使用CDN加载，需要联网访问
2. **浏览器兼容**：建议使用Chrome 90+、Firefox 88+、Edge 90+
3. **性能优化**：首次加载可能需要5-10秒（加载多个ECharts地图数据）
4. **iframe跨域**：如果部署到服务器，确保同源策略正确配置

## 🐛 常见问题

**Q: 页面显示空白？**
A: 检查浏览器控制台是否有CDN加载失败，可尝试刷新页面或更换网络。

**Q: 滚动不流畅？**
A: 确保浏览器硬件加速已开启，关闭不必要的浏览器扩展。

**Q: 移动端显示异常？**
A: 部分页面针对桌面端优化，移动端建议横屏查看。
